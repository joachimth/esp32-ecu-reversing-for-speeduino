OEM Ignition Logger

Flash din ESP32 direkte fra browseren.
Kræver Chrome eller Edge med Web Serial.

Efter flash

  1. Tilslut til WiFi: IgnLogger  /  ignition1
  2. Åbn: http://192.168.4.1
  3. OTA opdatering: http://192.168.4.1/update
  4. Kalibrering: Tryk GPIO0 ved idle (10° BTDC)

NE signal: 10k/20k spændingsdeler  |  IGT signal: 33k/10k spændingsdeler
